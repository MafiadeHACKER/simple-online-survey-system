# Simple Online Survey System Using HTML

# Description
This project is a simple Online Survey System created using only HTML. It allows users to fill in a short survey and view example responses in a table format. The project demonstrates the use of basic HTML elements such as forms, dropdowns, text areas, and tables.

# Features
    Collects user details: Name, Age, Favorite Color, and Feedback
    Displays sample survey responses in a table
    Built entirely with HTML5

# How to Use
    1. Download or clone the repository.
    2. Open the index.html file in your browser.
    3. Fill in the survey fields and explore the example responses.
